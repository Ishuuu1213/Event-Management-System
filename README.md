# Event-Management-System


## Structure:
- backend/: Node + Express server (serves API and frontend static files)
- frontend/: static HTML/CSS/JS pages: index, login, admin, vendor, user

## How to run locally:
1. cd event-management-project/backend
2. npm install
3. npm start
4. Open http://localhost:3000 in browser

## Demo credentials:
- Admin: username=admin password=admin

## Notes:
- This is a simplified demo following the provided flowchart (Index -> Login -> Admin/Vendor/User).
- Data stored in backend/data.json (simple JSON file).
- Extend as needed (use database, authentication, validation) for production.
